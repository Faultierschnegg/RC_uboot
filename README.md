In this project, I want to create a remote controlled (maybe even live in the water, but that comes later) RC-Submarine with as few expensive parts as possible.

For the first Prototyp, I use an old Tupperware container to first build a working boat and after being able to remote control this via ELRS using a standard gaming controller dive up to one meter down into the water.
I want to test if my idea using plastic syringes to mimic a swim bladder works and if and how I can somewhat accuratly hold the depth of my submarine.

Currently I am using an arduino nano r4 as my onboard computer and an 8bit do wireless as my controller.
